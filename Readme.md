# Rest API - Golang

## Model
- Notes
    ```json
    {
        "id": 123,
	    "title": "Golang",
	    "body" : "Golang adalah bahasa yang dibuat oleh google",
    }
    ```

# Endpoint
## Create Note
- URL : `localhost:8080/api/v1/notes`
- Method: `POST`
- request body
    ```json
    {
        "title": "Golang",
	    "body" : "Golang adalah bahasa yang dibuat oleh google",
    }
    ```
- response
    ```json
    {
        "code": 201,
	    "status": "Success",
	    "message" : "Note Baru Berhasil Ditambahkan",
    }
    ```
## List Note
- URL : `localhost:8080/api/v1/notes`
- Method: `POST`
- response
    ```json
    {
        "code": 200,
	    "status": "Success",
	    "message" : "Success get list notes",
        "data" : [
            {
                "id": 1,
                "title": "golang",
                "body": "bahasa golang dari google"
            },
            {
                "id": 2,
                "title": "js",
                "body": "javascript beda dari java"
            }
        ]
    }
    ```
